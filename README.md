# 🧠 AI Academic Integrity Checker

A Python-based tool that analyzes text to estimate whether it is **AI-generated or human-written**, using Groq API.

## 🚀 Features

* AI vs Human classification
* Probability estimation
* Reasoning + feedback
* Optional Streamlit UI

## ▶️ Run

### CLI
python app.py

### UI (Streamlit)
streamlit run ui.py

## ⚠️ Note

AI detection is **not 100% accurate** — results are probabilistic.
